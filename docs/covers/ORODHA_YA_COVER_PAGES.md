## JINSI YA KUTUMIA

1. **Kufungua Cover Page:**
   ```bash
   open /home/latest/docs/covers/[Cover_Page_Name].html
   ```

2. **Kuchapisha PDF:**
   - Fungua cover page katika browser
   - Print/Chapisha (Ctrl+P au Cmd+P)
   - Chagua "Save as PDF"
   - Hifadhi kwenye saraka unayotaka

3. **Kuunganisha na Nyaraka:**
   - Chapisha cover page kama PDF
   - Unganisha na nyaraka yako kwa kutumia PDF merger tools

## MAELEZO

Cover pages zote zimetengenezwa kiotomatiki kwa kutumia script ya Node.js iliyopo katika:
```
/home/latest/scripts/generate-cover-pages.js
```

Ikiwa unahitaji kutengeneza upya au kurekebisha cover pages, tekeleza:
