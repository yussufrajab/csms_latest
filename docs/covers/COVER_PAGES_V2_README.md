## 💻 JINSI YA KUTUMIA

### Kufungua Cover Page katika Browser

```bash
# Firefox
firefox /home/latest/docs/covers/Technical_Architecture_Document_Cover_Page_v2.html

# Chrome
google-chrome /home/latest/docs/covers/User_Manual_Cover_Page_v2.html

# Kutumia 'open' (ikiwa ipo)
open /home/latest/docs/covers/Security_Assessment_Report_Cover_Page_v2.html
```

### Kuchapisha kama PDF

1. Fungua cover page katika browser
2. Bonyeza **Ctrl+P** (Windows/Linux) au **Cmd+P** (Mac)
3. Katika dialog ya print:
   - Chagua destination: **"Save as PDF"**
   - Hakikisha "Background graphics" iko checked
   - Ukubwa wa ukurasa: **A4**
4. Click **Save**
5. Chagua mahali pa kuhifadhi

### Kuunganisha na Nyaraka Yako

Unaweza kutumia PDF tools kuunganisha cover page na nyaraka yako:

```bash
# Kwa kutumia pdftk
pdftk cover.pdf document.pdf cat output final.pdf

# Kwa kutumia pdfunite (poppler-utils)
pdfunite cover.pdf document.pdf final.pdf
```
